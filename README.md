# SkillSwap – A Local Skill Exchange Platform

## Purpose
SkillSwap is an interactive platform where users can offer, learn, and trade skills locally.  
For example, users can provide or learn guitar lessons, language exchange, coding help, or yoga training. Users can browse listings, rate experiences, and connect with local skill providers.


## Key Features
- User authentication with email & Google
- Browse and list local skills
- Book sessions with skill providers
- Interactive rating system
- Simple and responsive UI
- Toast notifications for actions

## NPM Packages Used
- `react-router-dom` – Routing
- `react-hot-toast` – Notifications
- `firebase` – Authentication & Database
- `react-icons` – Icons
- `axios` – HTTP requests (if any API calls are used)
- `tailwindcss` – Styling (if used)
